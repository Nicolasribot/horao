Installation

Source code

Horao source code is available on the corresponding github repository.

The master branch is the development branch and has a tag for each released version.

Supported platforms

Horao has been successfully compiled and tested on Linux 64 bits with gcc.

Requirements

  • A C++ compiler, see above for supported platforms
  • CMake version ≥ 2.8.6
  • OpenSceneGraph version ≥ 3.0.1
  • Boost version ≥ 1.46
  • PostGIS version ≥ 2.1
  • GDAL version ≥ 1.9

QGIS Plugin installation

Create a link in $HOME/.qgis2/python/plugins/ to horao/qgis_plugin:

ln -s horao/qgis_plugin $HOME/.qgis2/python/plugins/

The executable horaoViewer (generated by the compilation) must be in the PATH when launching qgis for the plugin to function.

Compilation

The compilation process is based on CMake, run:

cmake . && make && sudo make install

You may specify dependencies locations with environment variables in case cmake doesn't find them automatically, il particular, if you compile OpenSceneGraph from sources, the default install location on 64 bits platform is /usr/local/lib64, ans you must define OSG_ROOT as such before runnig cmake.

Default building options should work out-of-the-box. You may want to fine tune the compilation process with build options.

Varible Name Description
OSG_ROOTDirectory where libosg.so can be found
Build options (cmake options)
Option Name Default Description
CMAKE_INSTALL_PREFIX /usr/local Specifies where Horao will be installed when make install is invoked
CMAKE_BUILD_TYPE Release Switches between a Release build, specially crafted for speed efficiency and a Debug build, specifically crafted for development.
