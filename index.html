Current architecture

A simple viewer designed to listen to commands on its standard input. See doc/protocol.txt for the basic protocol specifications (still subject to changes)

A Python plugin that is used to connect QGIS signals to the viewer (in another process) to allow loading of QGIS layers with 3D geometries. The plugin depends on a QGIS branch where symbology signals have been added.
